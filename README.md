# Boilerplate NextJS

Estrutura de projeto já configurada com algumas bibliotecas e outros para trabalhar com NextJS.

## Tecnologias
- ReactjS
- NextJS
- Jest
- next-pwa
- styled-Components
- Typescript
- husky
- eslint
- prettier
- lint-staged

Observação: Se seu projeto não necessita de pwa, é só remover o `next-pwa` e limpar o arquivo `next.config.js`
